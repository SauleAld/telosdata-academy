Consulting & Outsourcing

For our Corporate Clients, we provide consulting and outsourcing services in software quality assurance.

Consulting: We provide professional services and corporate training for automated quality assurance with Selenium/Java, RestAssured/Java and custom testing tools development in various technology stacks.

Outsourced Managed Projects: We have the ability to ramp up a onshore/offshore QA and Automated Tests Delivery Team within days. Our project management and client engagement teams are based in America.

Case studies

For a large government financial institution, we developed Automated Testing framework in Selenium/Java with a custom application to support building automated tests in Java automatically without a lot of coding effort. The tests ran on VDI (Virtual Desktop Infrastructure) using Selenium Grid.

For a large electricity grid operator, we worked on their new application to stream line electricity capacity and generation source data in a real-time streaming REST application based on Java/Mongo DB (NoSQL) with front-end using ReactJS. We designed API and UI tests in Rapise (JS based automated testing tool) for the customers QA team. Application was designed with ReactJS and NodeJS.

For a restaurant and hospitality company based in Chicago specializing in providing vending and food services to major entertainment and sports venues, we worked on a project to migrate existing POS system to a custom Android based payment system. We setup API and UI testing framework in Java running tests on TFS server and Amazon Web Services cloud with a .NET/MS SQL application stack.
